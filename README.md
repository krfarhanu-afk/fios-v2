# FIOS - Financial Intelligence Overview System

## V1 Scope

This project is the V1 implementation of the Financial Intelligence Overview System (FIOS).

### Core Features:
- **Today Dashboard:** A high-level overview of the market and key tickers.
- **Tracked Tickers:** `AAPL`, `MSFT`, `NVDA`, `SPY`, `QQQ`.
- **Macro Page:** A dedicated page for macroeconomic indicators.
- **Research Vault:** A place to store and manage research notes.
- **AI Daily Brief:** An AI-generated summary of the day's financial news.
- **Data Caching:** All data is read from a local cache to ensure fast page loads and avoid API rate limits.
- **Password Protection:** The application is protected by a password login.
